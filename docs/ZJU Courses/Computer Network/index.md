# Intro

我所在的班级是 2025 年秋陆系群老师的非教改班，实验没有沿用教改班的 CS144，仍然使用老一套的 GNS3 实验。

课程使用的教材是 Andrew S. Tanenbaum 的 *Computer Networks, 5th. ed*。老师提供了第五版的英文原版。

清华大学出版社有中文翻译版，目前已经更新了最新的第六版的翻译本。笔者使用的是第六版的中文译本，发现第六版相较课堂使用的第五版有部分差别，但是总体感觉对学习影响不大。而且感觉第六版更适合自学一点。

**我十分推荐在上课而之前，先提前慢慢阅读教材的对应章节！Tanenbaum 的教材实际上写的十分有趣，同时也有中文版。你可以根据下面陆老师在笔者班级的教学进度进行规划：**

??? Tips "上课规划"
    - 第一周 Overview
    - 第二周 Physical Layer：数据传输理论基础，传输介质
    - 第三周 Physical Layer：数字调制，多路复用，通信系统实例
    - 第四周 Data Link Layer：framing，错误检测与纠正
    - 第五周 Data Link Layer：基本数据链路层协议，ARQ，流量控制，协议实例
    - 第六周 MAC Sublayer: Channel Allocation Problem，ALOHA，CSMA
    - 第七周 MAC Sublayer: Ethernet, 802.11 WLAN, 


与此同时，我参考了 James F. Kurose 的 *Computer Networking: A Top-Down Approach* 作为参考。这本书显然比 Tanenbaum 更适合自学一点，而且这也是你浙非计院的计网课使用的教材。并且，虽然你浙计院的计网课仍然使用自底向上的路线进行讲解，倒着读这本书来作为参考个人感觉也是完全没有什么影响的。**不过这本书对物理层的涉猎不多**，对物理层还是需要读 Tanenbaum 。
